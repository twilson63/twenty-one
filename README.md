# twenty-one - React Style

A React Blackjack Game using the deckofcardsapi.com

![playing-cards-1.jpg](playing-cards-1.jpg)

## Run

```
npm i serve -g
serve -c 0 -s
```

- components
  - app.js
  - controls.js
  - dealer.js
  - player.js
  - card.js
- store.js (state)
- actions.js (game logic)
- lib.js (utility)
- index.js (main)
